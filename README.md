# Book Ducks 
A bookstore web application where users can browse books, 
create an account to save books to a "Want to Read" list,
and rate books from 1-10. Admins can add new books directly from the website. 
Final project for the "Interaction with CMS" course at Nackademin.

## How to run
1. Navigate to the backend folder: `cd backend`
2. Install dependencies: `npm install`
3. Start Strapi: `npm run develop`
4. Strapi runs at http://localhost:1337
5. Open `frontend/index.html` in a browser (or use VS Code Live Server)
